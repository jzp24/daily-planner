# 日程本 — 设计规范

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #4A90D9 | 淡蓝，按钮、Tab 激活态、标题栏 |
| 主色浅 | #E8F1FB | 淡蓝背景、卡片底色 |
| 主色深 | #3A7BC8 | hover/按下状态 |
| 成功绿 | #52C41A | 年度热力图 ≥4h、完成状态 |
| 警告红 | #FF4D4F | 年度热力图不足、删除按钮 |
| 中性灰 | #F0F2F5 | 页面背景 |
| 文字主 | #333333 | 正文 |
| 文字次 | #999999 | 辅助说明 |
| 边框 | #E8E8E8 | 卡片/输入框边框 |

## 字体

- 系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- 中文备选：`"PingFang SC", "Microsoft YaHei"`
- 数字/代码：`"SF Mono", "Menlo", monospace`
- 基准字号：14px（移动端）/ 16px（桌面端）

## 间距

- 基础单位：8px
- 常用间距：8px / 12px / 16px / 24px / 32px
- 页面内边距：16px（移动端）/ 24px（桌面端）
- 卡片圆角：8px
- 按钮圆角：6px

## 响应式断点

| 断点 | 宽度 | 适配 |
|------|------|------|
| 移动端 | < 768px | 单列，底部 Tab 导航 |
| 桌面端 | ≥ 768px | 最大宽度 960px 居中，左侧导航 |

## 组件规范

### 按钮
- 主按钮：淡蓝背景 + 白色文字
- 次要按钮：白底 + 淡蓝边框 + 淡蓝文字
- 危险按钮：白底 + 红色边框 + 红色文字
- 最小触控区域：44×44px（移动端）

### 输入框
- 边框 1px solid #E8E8E8
- focus 态边框改为淡蓝色
- 圆角 6px，内边距 8px 12px

### 卡片
- 白底 + 1px 边框 + 8px 圆角
- 内边距 16px
- 微阴影：0 1px 3px rgba(0,0,0,0.06)

### 模态框
- 居中弹出，背景半透明遮罩 rgba(0,0,0,0.45)
- 内容区白底圆角，最大宽度 400px
