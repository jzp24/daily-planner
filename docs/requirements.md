# 日程本 — 需求文档

## 项目定位

本地浏览器端日程管理单页 Web 应用。数据存于浏览器 localStorage，无需后端、无需注册。电脑和手机浏览器均可使用。

## 功能需求

### F1 每日待办管理

- 以天为单位，日历选日查看/管理当天任务
- 任务字段：标题、所属分类、开始时间、结束时间、状态
- 计时方式：支持手动输入时间 + 点按钮自动计时
- 显示当天所有任务总耗时

### F2 倒计时

- 可添加多条倒计时（标题 + 目标日期）
- 显示"距离 [标题] 还有 [N] 天"

### F3 周目标 & 周总结

- 以周一至周日为一个周期
- 每周可设置多条周目标
- 每周可写周总结/反思（自由文本）

### F4 全年总览图（热力图）

- 类似 GitHub 贡献图，每天一个小格子
- 颜色规则（按当天所有任务总时长）：
  - ≥ 4 小时 → 绿色
  - 3 分钟 < 时长 < 4 小时 → 红色
  - ≤ 3 分钟或无任务 → 灰色/无填充

### F5 时长统计

- 折线图：选定一周内每天专注时长趋势
- 饼状图：选定时间段内各大分类时长占比
- 日期范围统计：日均专注时长 + 总专注时长

### F6 UI 要求

- 风格：简洁、直观
- 主色调：淡蓝色 (#4A90D9 系)
- 响应式布局：桌面端 + 移动端
- 5 个底部 Tab：每日 / 每周 / 统计 / 年览 / 设置

## 非功能需求

- 纯前端，无后端依赖
- 数据本地存储，刷新不丢失
- 首次加载无需等待
- 移动端触控友好
